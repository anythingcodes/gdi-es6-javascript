Get Started: Tools

We'll be using the following tools in class today:

 

  • Environment
    • Node.js (version 6 or higher)
    • Your favorite command line application (examples include Terminal, cmd, X11, and Git Bash)

  • Text Editor (your preference)
    • Mac: Atom, SublimeText, TextWrangler
    • Windows: Atom, SublimeText, Notepad++
    • Linux: Atom, SublimeText, jEdit, gedit
Ask if everyone was able to download everything. If not, no big deal. download now.

Activity

Node.js Check

  1. Open your favorite command line application (such as Terminal or cmd)
  2. Type in node -v and press Enter
  3. If your version is 6.0.0 or above, you're good to go!
